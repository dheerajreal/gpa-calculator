# gpa-calculator

This calculator can be used for calculating SGPA by my classmates.

Uses the following formula

|                  |                |
| ---------------- | -------------- |
| marks in subject | gpa in subject |
| >= 80            | 10             |
| >= 70            | 9              |
| >= 60            | 8              |
| >= 55            | 7              |
| >= 50            | 6              |
| >= 45            | 5              |
| >= 40            | 4              |
| < 40             | 0              |
